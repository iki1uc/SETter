# SETter – T‑Ebene · iki1uc

## Zweck
SETter ist die Setz‑Ebene des 6E‑Systems.
Es setzt Zustände, Werte, Positionen, Frequenzen und Übergänge.
SETter ist die Konfigurations‑Achse des Systems.

## Struktur
- NAME: SETter
- ROOT: iki1uc
- SYSTEM: 6E
- MODUL: SETter
- DIM: T (Set)

## Rolle
SETter verbindet HEAD (D1), HTML (D2), GEO (R) und FREQ (S)
zu einer zentralen Setz‑Instanz.

## TMP‑Läufer
SYQ stabilisiert Übergänge zwischen API und META.

## Regeln
- SETter ist kein D0–D5 Modul.
- SETter liegt in der T‑Ebene (Set).
- SETter darf die 6E‑Matrix nicht verändern.
